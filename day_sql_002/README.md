# Day 2 - MySQL Practice

## Task 1: Create Table

Create a table named `employees` with the following columns:

- emp_id (INT, Primary Key)
- emp_name (VARCHAR)
- department (VARCHAR)
- salary (INT)

## Task 2: Insert Data

Insert sample employee records into the table.

## Task 3: Select All Records

Display all records from the `employees` table.

## Task 4: WHERE Condition

Fetch employees who belong to the IT department.

## Task 5: Salary Filter

Fetch employees whose salary is greater than 40000.

## Learning Outcome

- CREATE TABLE
- INSERT INTO
- SELECT
- WHERE condition
