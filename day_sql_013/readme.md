# Day 13 - MySQL Practice

## Topic: VIEWS

## Task 1: Create Table

Create a table named `employees` with:

- emp_id (INT, Primary Key)
- emp_name (VARCHAR)
- department (VARCHAR)
- salary (INT)

## Task 2: Insert Data

Insert sample employee records.

## Task 3: CREATE VIEW

Create a view `it_employees` that shows only IT department employees.

## Task 4: SELECT from VIEW

Fetch data using the created view.

## Task 5: UPDATE using VIEW

Update salary of an employee using the view.

## Task 6: DROP VIEW

Remove the view from the database.

## Key Concepts

- VIEW is a virtual table
- Used for security and simplified queries
- Data can be updated through simple views

## Learning Outcome

- CREATE VIEW
- SELECT from VIEW
- UPDATE VIEW
- DROP VIEW
