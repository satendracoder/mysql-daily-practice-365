# Day 7 - MySQL Practice

## Task 1: Create Tables
Create two tables:
- `customers`
- `orders`

## Task 2: Insert Data
Insert sample records into both tables.

## Task 3: INNER JOIN
Fetch customer name, city, product, and amount by joining
`customers` and `orders` tables using `customer_id`.

## Key Concept
- INNER JOIN returns only matching records from both tables.

## Learning Outcome
- INNER JOIN
- Table alias
