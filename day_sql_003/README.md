# Day 3 - MySQL Practice

## Task 1: Create Table

Create a table named `products` with:

- product_id (INT, Primary Key)
- product_name (VARCHAR)
- category (VARCHAR)
- price (INT)

## Task 2: Insert Data

Insert sample product records into the table.

## Task 3: Select All Products

Display all records from the `products` table.

## Task 4: DISTINCT Keyword

Fetch unique product categories.

## Task 5: ORDER BY

Display products sorted by price in descending order.

## Task 6: LIMIT

Fetch the top 3 most expensive products.

## Learning Outcome

- DISTINCT
- ORDER BY
- LIMIT
