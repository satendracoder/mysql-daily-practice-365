# Day 5 - MySQL Practice

## Task 1: Create Table
Create a table named `orders` with:
- order_id (INT, Primary Key)
- customer_name (VARCHAR)
- category (VARCHAR)
- amount (INT)

## Task 2: Insert Data
Insert sample order records into the table.

## Task 3: COUNT Function
Count total number of orders.

## Task 4: SUM Function
Calculate total order amount.

## Task 5: AVG Function
Calculate average order value.

## Task 6: MIN and MAX
Find minimum and maximum order amounts.

## Task 7: GROUP BY
Calculate total sales category-wise.

## Learning Outcome
- COUNT()
- SUM()
- AVG()
- MIN()
- MAX()
- GROUP BY
