# MySQL Daily Practice 365

## Overview

This repository is designed for **daily MySQL practice** for **365 days**.  
It helps you strengthen your database skills step by step, from **basic SQL queries** to **advanced MySQL concepts**.

Each day contains:

- A dedicated folder named `day_sql_XXX` (e.g., `day_sql_001` for Day 1)
- `problem.sql` → SQL queries for the day’s tasks
- `README.md` → Problem description and hints for reference
